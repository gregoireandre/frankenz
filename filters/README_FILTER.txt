- FILTER files (e.g., irac_ch1.res) should be in a two-column format [wavelength (angstroms), transmissivity (0 to 1)] separated by a space.
- LIST files (e.g. SPLASH_mod.list) should be in a two-column format [shorthand name (arbitrary), filename (local path)] separated by a space.

- All LIST and FILTER files should be contained within this (FILTER/) folder.
